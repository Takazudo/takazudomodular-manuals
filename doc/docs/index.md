# Takazudo Modular: Manuals Documentation

マニュアルビューアープロジェクトの開発ドキュメントへようこそ。

## ドキュメントカテゴリ

### INBOX

プロジェクトの主要な開発ドキュメントが含まれています:

- [Design System](inbox/design-system.md) - Zudo Design Systemの詳細説明

## プロジェクト概要

このプロジェクトは、ハードウェアシンセサイザーのマニュアルビューアーです。複数のマニュアルをバイリンガル形式（英語のPDFページ + 日本語翻訳）で提供します。

## 技術スタック

- **Next.js 14+** - サーバーサイドレンダリングと静的生成
- **React 19** - UIフレームワーク
- **TypeScript** - 型安全性
- **Tailwind CSS v4** - Zudo Design Systemを使用したスタイリング
- **Docusaurus 3** - 技術ドキュメントシステム（本サイト）
